Compiling from source on Linux

Important

Regular users are recommended to download an install a release version, rather than compiling from source. Releases are published at https://github.com/axoloti/axoloti/releases. Assuming Ubuntu Linux. Other Distributions may need minor changes.

Cloning the GIT repository

Open a terminal and enter:

$> git clone https://github.com/axoloti/axoloti.git
$> cd axoloti
$> git submodule update --init --recursive

Compiling

open a terminal and enter:

$> cd platform_linux
$> ./build.sh
$> cd ..

Root privileges are required to add a udev rule to grant access to Axoloti Core on USB. The install script will ask you for this.

Launching

$> ./Axoloti.sh
